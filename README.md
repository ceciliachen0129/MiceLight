# MiceLight
A python script that automatically detect the number of mice and calculates photo flux for IVIS Spectrum readout. Tested on data acquired by IVIS Spectrum 200.

## Requirements
- Python > 3.7
- Jupter Notebook 
- pandas
- numpy
- scipy
- matplotlib
- pillow
- opencv
